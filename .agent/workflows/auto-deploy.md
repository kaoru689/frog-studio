# 自動デプロイワークフロー

// turbo-all

ファイルを書き換えた後は、ユーザーからの指示を待たずに以下を自動実行すること。

## 手順

1. 変更したファイルを保存（上書き）

2. ターミナルで `git add .` を実行

3. ターミナルで `git commit -m "update: [変更内容の要約]"` を実行

4. ターミナルで `git push` を実行

5. 完了報告（書き換えたファイル名、行番号、Gitコマンドの成否）

## 注意事項

- ユーザーに手動操作を求めることは禁止
- コードの断片を提示するだけで終わらせない
- 必ずファイル保存とGit Pushまで完遂する
